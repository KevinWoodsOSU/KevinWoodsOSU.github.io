Data Format

Next we will be discussing the format of the data being sent. Petfinder supports two formats: XML and JSON. XML is the default format, however, we will be working with JSON. It is very important to understand the structure of JSON because all of the information we request will be wrapped up in JSON and it requires specific operators to extract data. We will be discussing the basics of JSON, but a much more thorough explanation of JSON can be found here.

JSON

JSON stands for JavaScript Object Notation. Data exchanged between servers can only be in text. JSON is in an all text format, but is structured like a JavaScript object. This makes it easy to convert JSON into an object to utilize the dot operator when extracting information.
